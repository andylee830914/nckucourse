NCKU Portable Schedule

View project on GitHub

About

This is an application write in swift. It can automatically capture the course data from school course catalog and save on your iPhone. With this, you can have your own daily course schedule in your pocket.

Data source

All the data is from the NCKU course catalog instantly when you add.
